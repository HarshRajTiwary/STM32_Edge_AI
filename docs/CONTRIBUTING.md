# Contributing to STM32 Edge AI

Thank you for improving this project! This file describes how to contribute, report issues, and submit enhancements.

## How to contribute
1. Fork the repository on GitHub.
2. Clone your fork:
   ```bash
   git clone https://github.com/<your-username>/STM32_Edge_AI.git
   ```
3. Create a new branch for your feature or fix:
   ```bash
   git checkout -b feature/my-improvement
   ```
4. Make changes and run local tests or sample commands.
5. Commit with a clear message.
6. Push to your fork and open a pull request.

## Contribution guidelines
- Keep code style consistent with existing Python scripts.
- Add documentation for new features and scripts.
- If adding a new model type, include a README example and a run command.
- Keep functions small and easy to understand.

## Reporting issues
- Use GitHub Issues to report bugs, missing documentation, or improvement ideas.
- Include steps to reproduce, error messages, and environment details.

## Project structure
- `README.md` contains usage and tutorials.
- `src/` contains code for data, training, quantization, and deployment.
- `docs/` contains supporting docs and tutorials.
- `stm32/` contains STM32-specific deployment information.
