# STM32 Edge AI Project Overview

This project is an end-to-end pipeline for training machine learning models on a PC and deploying them on STM32 microcontrollers.

## Why this project exists
- STM32 microcontrollers are widely used in embedded systems.
- Embedded AI enables smart sensor processing, anomaly detection, gesture recognition, and predictive maintenance.
- This repository bridges desktop ML workflows to STM32 deployment using TensorFlow Lite for Microcontrollers (TFLM) and STM32Cube.AI.

## What this repository contains
- Data preparation and synthetic dataset generation
- Preprocessing utilities and data pipeline helpers
- Multiple STM32-friendly model architectures
- Training and model export scripts
- TFLite quantization and C source conversion helpers
- STM32 deployment guidance and project-ready format

## Target users
- beginners with little or no embedded ML experience
- students learning model deployment on microcontrollers
- developers building proof-of-concept AI on STM32
- anyone who wants a complete pipeline from data to device
